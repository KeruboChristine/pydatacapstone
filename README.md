
Youth Unemployment Crisis in Kenya 

 Overview

This project explores the youth unemployment crisis in Kenya, analyzing trends, identifying patterns, and investigating key factors that contribute to high unemployment rates among Kenyan youth. Using publicly available data, the goal is to provide insights that can inform policy, education, and employment strategies.

The analysis is done using Python, focusing on data cleaning, visualization, and statistical exploration.

Data Source

The dataset is sourced from The World Bank:

* [World Bank Kenya Youth Unemployment Data](https://data.worldbank.org/indicator/SL.UEM.1524.ZS?locations=KE)
  
The data requires some cleaning and preprocessing.

Research Questions

This project seeks to answer the following:

1. What is the trend of youth unemployment in Kenya over the past decades?
2. What are the possible economic, educational, and social factors influencing youth unemployment?
3. Can we identify correlations between youth unemployment and other socio-economic indicators such as GDP, education, or labor force participation?

World Bank Kenya Youth Unemployment Data
Time Frame

The data covers youth unemployment in Kenya from 1991 to 2023, providing a longitudinal view of how youth unemployment has evolved over the last three decades.

Data Description

The dataset contains:

Youth Unemployment Rate (% of total youth labor force) – primary indicator
Breakdown by gender – male and female youth unemployment rates
Related socio-economic indicators – labor force participation, GDP per capita, education levels (optional for correlation analysis)


Key Features

* Data cleaning and preprocessing of World Bank datasets
* Trend analysis of youth unemployment in Kenya
* Visualization using Python libraries (Matplotlib, Seaborn, Plotly)
* Exploratory data analysis and correlation insights
* Potential recommendations for policymakers

 Technologies & Libraries

* Python 3.x
* Pandas, NumPy (data manipulation)
* Matplotlib, Seaborn, Plotly (visualization)
* Jupyter Notebook (analysis)
